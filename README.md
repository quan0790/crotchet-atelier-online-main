## Getting Started

You can edit and run this project locally using Node.js and npm.

### Prerequisites

* Node.js & npm installed (recommended via [nvm](https://github.com/nvm-sh/nvm))

### Steps to Run Locally

```bash
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
```

Your development server will auto-reload and provide an instant preview.

## Deployment

To deploy the project, build it locally:

```bash
npm run build
```

Then serve the `dist` folder using your preferred hosting service (Netlify, Vercel, or any static server).
